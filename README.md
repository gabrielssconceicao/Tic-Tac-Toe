## ♻️ How Contribute

```bash
# Clone the repository
$ git clone https://github.com/gabrielssconceicao/Tic-Tac-Toe.git
```

```bash
# Enter directory
$ cd Tic-Tac-Toe
```

```bash
# Installing dependencies, if npm
$ npm install
```

<p align="center">or</p>

```bash
# Installing dependencies
$ yarn
```

## Template

<code>npm create vite@latest . --template react</code>
